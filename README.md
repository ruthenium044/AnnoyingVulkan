# Vulkan

Mini project following tutorials and learning new things about graphics programming

## Controls:

moveLeft = A;

moveRight = D;

moveForward = W;

moveBackward = S;

moveUp = E;

moveDown = Q;

lookLeft = Arrow left;

lookRight = Arrow right;

lookUp = Arrow up;

lookDown = Arrow down;
